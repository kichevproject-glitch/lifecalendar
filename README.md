# LifeCalendar 🗓

A personal interactive calendar with categories, photo attachments, sharing, and AI analytics.

## Setup

### 1. Environment Variables
In Vercel, add these environment variables:
```
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

### 2. Deploy
This project is deployed automatically via Vercel when you push to GitHub.

## Tech Stack
- React + Vite
- Supabase (database, auth, storage)
- Vercel (hosting)
- date-fns (date utilities)
